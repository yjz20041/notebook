#### 2.line height

```js

1. 根据各自的font-size和line-height计算出每一个inline box的高度，font-size决定content-area高度，
然后上下各加上 (line-height - font-size) / 2
2. 根据vertical-align排好inline box，其中最高的inline box的上边到最低inline box的下边就位 line height。
```
